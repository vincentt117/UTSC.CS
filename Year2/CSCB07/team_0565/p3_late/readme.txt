Running the Application:
Through terminal/cmd, cd through this path in the project
 - cd p3_late/CSCB07CourseProject/app/src/main/java/sampletests
With adb push into the sd card the following files (and additional client/flight info .txt files):
 - clientstorage.ser
 - flightstorage.ser
On the emulator/android deviceL 
 - Settings > App Permission > Unrestricted Data Usage > *Grant permission to FlightBook
Run the app: a pop-up will appear asking if you wish to allow the app to access external storage on your device. Please allow it.
